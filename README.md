lein# Predictive Maintenance using Clojure

## Project Goal
Build a simple ML system to predict machine failure using sensor data.

## Steps to Run

1. Install Leiningen (https://leiningen.org/)
2. Download AI4I 2020 Predictive Maintenance dataset (CSV format)
3. Place the file inside the data/ folder and rename it to:

   ai4i2020.csv

4. Open terminal inside project folder
5. Run:

   lein run

You should see dataset rows printed on screen.

Next step will be adding Decision Tree training.
